### [1.0.0-alpha](https://github.com/gabrielnsw/network-topology-plugin/tree/1.0.0-alpha) (2026-02-23)

**Implemented enhancements:**

- Introduce Cytoscape.js core engine for interactive visual mapping
- Direct data integration with Alexander Zobnin's Zabbix DataSource DataFrames
- Interactive drag-and-drop modals for adding/editing devices and links
- Add support for dark and light Grafana themes mapping
- Implement backup and restore topology state functionality via JSON
- Native internationalization support featuring English, Spanish, and Portuguese translation
